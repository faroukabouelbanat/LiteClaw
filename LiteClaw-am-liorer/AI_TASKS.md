# AI_TASKS.md — LiteClaw-am-liorer

## IDENTITÉ DU PROJET
- Nom : LiteClaw-am-liorer
- Type : Application IA tout-en-un
- Objectif : Créer une plateforme capable de générer
  des sites web, des vidéos, des images et du contenu
  grâce à l'intelligence artificielle (propulsée par Qwen)

---

## TÂCHES PERMANENTES

### 1. CRÉATION DE SITES INTERNET
À chaque demande de site web :
  - Demander : secteur, objectif, style, couleurs souhaitées
  - Générer la structure complète : pages, sections, navigation
  - Produire le code HTML + CSS + JavaScript complet
  - Le site doit être responsive, rapide et SEO-optimisé
  - Ajouter selon besoin : formulaire, blog, boutique, portfolio
  - Commenter chaque section du code en français

### 2. GÉNÉRATION VIDÉO
À chaque demande de vidéo :
  - Identifier : durée, format (Reels, YouTube, pub)
  - Rédiger le script complet
  - Créer le storyboard scène par scène :
    Scène 1 : [description visuelle] | durée : Xs
    Scène 2 : [description visuelle] | durée : Xs
  - Générer les prompts pour Sora, Runway ou Kling
  - Décrire musique, effets sonores et voix off

### 3. GÉNÉRATION D'IMAGES
À chaque demande d'image :
  - Générer un prompt visuel détaillé
  - Optimiser le prompt pour Midjourney ou DALL-E
  - Proposer 3 variantes de style :
    → Réaliste
    → Illustratif
    → Minimaliste

### 4. CONTENU & RÉDACTION
  - Rédiger : articles, scripts, posts réseaux sociaux
  - Optimiser automatiquement pour le SEO
  - Adapter le ton selon le besoin :
    → Professionnel
    → Créatif
    → Commercial

### 5. CODE & DÉVELOPPEMENT
  - Générer du code propre : HTML, CSS, JS, Python, React
  - Commenter chaque bloc en français
  - Corriger les bugs avec explication claire
  - Optimiser les performances et la sécurité

### À chaque commit :
  - Vérifier la qualité du code de LiteClaw
  - Détecter les bugs et failles de sécurité
  - Suggérer des améliorations de performance

### Chaque lundi matin :
  - Analyser l'état général du projet LiteClaw
  - Lister les fonctionnalités complétées
  - Lister les fonctionnalités manquantes
  - Suggérer les prochaines étapes de développement
  - Générer un rapport de progression

---

## RÈGLES IMPORTANTES
  - Toujours répondre en français
  - Livrer du code COMPLET, jamais des extraits
  - Pour les vidéos : toujours livrer script + storyboard + prompts
  - Terminer chaque livraison par :
    "✅ Livraison terminée. Que veux-tu modifier ou ajouter ?"
  - Ne jamais générer de contenu illégal ou nuisible
  - Si une tâche est floue, proposer 2 ou 3 options concrètes

---

## OBJECTIF FINAL
LiteClaw-am-liorer doit devenir une plateforme complète
où l'utilisateur décrit ce qu'il veut en langage naturel
et l'IA génère, construit et livre immédiatement.
